# QR Attendance System

A full-stack web application for managing student attendance using QR code technology. The system enables faculty to generate QR codes for attendance sessions, students to mark attendance by scanning the QR code, and administrators to manage attendance records efficiently.

---

## 📌 Project Description

The QR Attendance System is designed to simplify the attendance process in educational institutions. It replaces traditional manual attendance with a secure, fast, and paperless QR code-based system. The application allows faculty to generate attendance sessions, record student attendance, and monitor attendance reports through an easy-to-use web interface.

---

## ✨ Features

- Secure Admin and Faculty Login
- Student Management
- QR Code Generation
- QR Code-Based Attendance
- Attendance Reports
- Search and Filter Records
- JWT Authentication
